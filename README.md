# Docker-контейнер для Redis

![Redis Logo](https://redis.io/images/redis-white.png)

> **Redis** (Remote Dictionary Server) — это высокопроизводительная база данных типа «ключ-значение» в оперативной памяти (in-memory), которая может использоваться как кэш, брокер сообщений и хранилище данных.

## Краткое описание и назначение

Данный проект предоставляет готовый Docker-образ для быстрого развертывания сервера **Redis**. Контейнер идеально подходит для разработки, тестирования и production-сред, где требуются быстрые операции с данными.

**Основные назначения контейнера:**
*   Кэширование данных для ускорения веб-приложений.
*   Хранение сессий пользователей.
*   Использование в качестве брокера для таких систем, как Celery.
*   Создание быстрых очередей сообщений.

---
docker run -d \
  --name my-secure-redis \
  -p 6380:6379 \
  -e REDIS_PASSWORD="my_secure_password" \
  -e REDIS_MAXMEMORY="512mb" \
  redis:latest

## Поддерживаемые аргументы образа (Environment Variables)

Для гибкой настройки Redis при запуске контейнера поддерживаются следующие переменные окружения:

| Переменная окружения | Описание                                                                 | Значение по умолчанию |
| -------------------- | ------------------------------------------------------------------------ | --------------------- |
| `REDIS_PORT`         | Порт, на котором будет работать сервер Redis.                            | `6379`                |
| `REDIS_PASSWORD`     | Пароль для аутентификации на сервере Redis. **Оставьте пустым для отключения.** | (пусто)               |
| `REDIS_DATABASES`    | Количество доступных баз данных.                                         | `16`                  |
| `REDIS_appendonly`   | Включение режима сохранения данных на диск (`appendonly`).               | `yes`                 |
| `REDIS_MAXMEMORY`    | Максимальный объем памяти, который может использовать Redis (например, `256mb`). | `0` (без ограничений) |
